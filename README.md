1️⃣ What is the difference between var, let, and const?
var is function-scoped and can be re-declared, let is block-scoped and cannot be re-declared but its value can change, while const is block-scoped and cannot be re-declared or reassigned.

2️⃣ What is the spread operator (...)?
The spread operator ... is used to expand or copy elements of an array or properties of an object.

3️⃣ What is the difference between map(), filter(), and forEach()?
map() returns a new array after transforming elements, filter() returns a new array with elements that pass a condition, and forEach() only loops through elements without returning a new array.

4️⃣ What is an arrow function?
An arrow function (=>) is a shorter syntax for writing functions in JavaScript.
ex:()=>{} this is arrow function.

5️⃣ What are template literals?
Template literals use backticks (`) and allow embedding variables or expressions inside strings using ${}.
